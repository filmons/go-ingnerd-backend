    Create Todo (POST):
        Set the request type to POST.
        Enter the URL http://localhost:3333/todo.
        Add a JSON body with the todo details.
        Send the request.

    Get All Todos (GET):
        Set the request type to GET.
        Enter the URL http://localhost:3333/todo.
        Send the request.

    Update Todo (PUT):
        Set the request type to PUT.
        Enter the URL http://localhost:3333/todo/:idTodo where :idTodo is the ID of the todo you want to update.
        Add a JSON body with the updated todo details.
        Send the request.

    Delete Todo (DELETE):
        Set the request type to DELETE.
        Enter the URL http://localhost:3333/todo/:idTodo where :idTodo is the ID of the todo you want to delete.
        Send the request.
